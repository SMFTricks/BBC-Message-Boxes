[size=12pt][color=navy]Introduction[/color][/size]
[size=8pt]BBC Message Boxes mod will allow you to use four different message boxes (error, warning, okay and info)[/size]

[size=12pt][color=teal]Features[/color][/size] [size=10pt][color=teal](Version 3.0+)[/color][/size]
[list]
[li][size=8pt]Select the type of the icons (classic, modern, modern octagone)[/size][/li]
[li][size=8pt]Set permissions for who can see these BBC icons in the BBC Editor.[/size][/li]
[/list]
